Features

Everything you need to stay compliant

Automated checks mapped to official ENS controls, with flexible output for any workflow.

🛡️

ENS / CCN-STIC 887

Controls directly mapped to the official Spanish National Security Scheme profile for AWS.

⚡

GraalVM native image

Single binary, no JVM required at runtime. Fast startup and zero dependencies to distribute.

🔁

CI/CD ready

Semantic exit codes and file-based reports designed for GitHub Actions and GitLab CI pipelines.

📄

Multiple output formats

Export results as table, JSON, HTML, or CSV — pick what fits your workflow.

🧩

Declarative controls

Extend the control set via ens-controls.yaml without touching a single line of code.

🔍

Granular filtering

Scope audits by severity (CRITICAL, HIGH…) or run a single named control.

Commands & exit codes

All commands, options, and semantic exit codes for scripting and pipeline integration.

CommandDescription
auditRun compliance audit on AWS infrastructure
audit -o json -f report.jsonSave audit report to a file in JSON format
audit -s CRITICALFilter results by minimum severity level
audit -c <control-id>Evaluate a single specific control
list-controlsList all available ENS controls with metadata
statusShow audit status and compliance summary
generate-completionGenerate bash/zsh shell completion script
--helpShow help for any command
Exit codeMeaning
0All controls compliant — audit passed
1One or more controls non-compliant
10Error fetching AWS resources
11Error evaluating controls
12AWS SDK / cloud error
Get started

Installation

Download a pre-built native binary for your platform or build from source.

📦
Download pre-built binary
Linux · macOS · Windows
# 1. Download the binary for your platform curl -LO https://github.com/juanmorschrott/ens-auditor/releases/latest/download/ens-auditor-linux-amd64 # 2. Make it executable and add to PATH chmod +x ens-auditor-linux-amd64 && mv ens-auditor-linux-amd64 /usr/local/bin/ens-auditor # 3. Configure AWS credentials export AWS_ACCESS_KEY_ID=... AWS_SECRET_ACCESS_KEY=... AWS_REGION=eu-west-1 # 4. Run your first audit ens-auditor audit
🔨
Build from source
Java 21+ · Maven 3.9+
git clone https://github.com/juanmorschrott/ens-auditor.git && cd ens-auditor ./mvnw clean install java -jar target/ens-auditor-0.1.0-SNAPSHOT.jar audit # Or build a native image with GraalVM ./mvnw -Pnative package -DskipTests && ./target/ens-auditor audit
🔁
GitHub Actions
CI/CD integration
- name: Run ENS Audit run: ./ens-auditor audit --output json --output-file ens-report.json - name: Upload report if: always() uses: actions/upload-artifact@v4 with: name: ens-compliance-report path: ens-report.json
